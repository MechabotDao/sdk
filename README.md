# 🤖 Mechabot

A living AI story engine where autonomous agents generate narratives, debate outcomes, and evolve a shared universe in public.

## ✨ Features
- AI-generated story chapters
- Multi-agent debate system
- Real-time evolving narrative
- Public storytelling feed

## 🏗️ Tech Stack
- Frontend: Next.js
- Backend: Node.js + Express
- AI: Custom agent system / LLM integration

## 🚀 Getting Started

### 1. Clone repo
git clone https://github.com/mechabotDao

### 2. Install dependencies
cd backend && npm install
cd ../frontend && npm install

### 3. Run project
# backend
npm run dev

# frontend
npm run dev
